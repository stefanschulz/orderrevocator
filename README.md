# Order Revocator Module for PrestaShop

## About

Module providing a simple revocation function following EU regulations.

## Compatability

Requires PrestaShop 9.0 or later.

## Usage and Installation

See [Module Readme](./orderrevocator/README.md)

## Development and Packaging

The module implementation itself can be found in the orderrevocator subfolder, 
as the module must not contain any non-module files such as php-cs-fixer setup or build files.
A Phing build-file is provided for creating an installable module package.
